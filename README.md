## Install
```sh
pnpm i
```
## Develop
### Run Env.
```sh
pnpm dev
```
### Prod. Build
```sh
pnpm build
```
### Preview Prod. Build
```sh
pnpm preview
```
### Format Code (ESLint + Stylelint + Prettier)
```sh
pnpm format
```
## Test
### Run Watch
```sh
pnpm test
```
### Coverage Report
```sh
pnpm cover
```
## Deploy
```sh
pnpm deployer
```
